**Условия соревнования:**
- Компбютер загадывает число от 1 до 100. Написанная программа угадывает его;
- Алгорит решения опирается на информацию о том, больше или меньше загаданное число, чем число из последнего "предположения".

**Метрика качества:**
- Результат работы программы оценивается по сренему числу попыток угадать число при 1000 повторений.
